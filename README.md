# Letters Calculator

A simple, interactive React application that calculates the number of characters in a phrase in real-time as the user types. This project focuses on managing **controlled components** and **state synchronization** in React.

## 🚀 Demo

## ✨ Features

  * **Real-time Calculation:** The letter count updates instantly with every keystroke.
  * **Controlled Input:** Utilizes React state to manage the input field value.
  * **Responsive Design:** Styled using CSS Flexbox for a clean, professional layout.
  * **User-Friendly UI:** Includes clear labeling and a highlighted result display.

## 🛠️ Tech Stack

  * **Frontend:** React.js
  * **Styling:** CSS3 (Flexbox, Linear Gradients)
  * **Assets:** CCBP Branding Assets

## ⚙️ Installation & Setup

To get this project running locally, follow these steps:

1.  **Clone the repository**

    ```bash
    git clone https://github.com/Sarvesh-Mohite/Character-Counter.git
    ```

2.  **Navigate to the project directory**

    ```bash
    cd count-characters-app-rjs
    ```

3.  **Install dependencies**

    ```bash
    npm install
    ```

4.  **Start the development server**

    ```bash
    npm start
    ```

    The app will run on `http://localhost:3000`.

## 🧠 What I Learned

  * How to handle **synthetic events** in React (specifically `onChange`).
  * Implementing the **State-View pattern** where the UI automatically reflects data changes.
  * Using the `htmlFor` attribute in JSX to link labels with inputs for better accessibility.
